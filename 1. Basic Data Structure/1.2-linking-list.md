# 1.2 Linking List

主要是在考指針的修改 or 使用Linking List的結構減少複雜度

## 1 Linking list vs Array 時間複雜度

* 新增刪除資料比較方便，但會需要比較多的儲存空間

|                               | linkling list | Array |
| ----------------------------- | ------------- | ----- |
| 查詢特定index資料             | O(n)          | O(1)  |
| 新增/刪除 資料                | O(n)          | O(n)  |
| 新增/刪除 資料 (已知節點位子) | O(1)          | O(n)  |
